Darklup - WP Dark Mode

Change into a Heart-throbbing dark mode interface of your WordPress website & Dashboard with automatic color conversion functionality. An all-in-one WordPress plugin to create a stunning dark version for your website.

Dark Mode Switches

Eccentric dark mode switch styles let users turn on/off dark mode with tap & slide.

Dark Based Image

Darklup lets you adjust image properties with a force-configuration method and changes the opacity, brightness, and contrast to match your dark mode setting. Supports all kinds of websites in WordPress.

Remember Last Mode

Stores previous dark mode data and lets the user activate it. Data contains custom presets, color customizations, switch styles, custom CSS, etc.

Frontend Dark Mode

Let users initiate dark mode by providing user-control switches. Automatically lets users interfere with the dark mode upon activation.

Backend Dark Mode

Activating this feature makes the Dark Mode switch appear on the WordPress backend for admin/users.

Replace Site Logo

Set dark mode logo by injecting logo URL in the settings field for dark mode.
